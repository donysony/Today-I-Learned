## [연산자] ==와 ===의 차이점

== : Equal Operator

=== : Strict Equals Operator

- a==b 라고 할 때,

a와 b의 값이 같은지를 비교 →  같음: true, 다름 : false (값만 같으면 true)

- a===b 라고 할 때,

값과 값의 데이터 타입이 모두 같은지 비교 → 같으면 true, 다르면 false
